## Reporting a Vulnerability

If you find a Vulnerability, open a Ticket in our [Discord Server](https://discord.com/invite/rx82u93hGD) or open a security advisory draft using [THIS](https://github.com/Aperture-OS/blink-package-manager/security/advisories/new) link.
Make sure you include examples and how to recreate the vulnerability.
